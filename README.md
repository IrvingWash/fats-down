# Fat's Down
## An app to brag about how you lose (or gain) your weight

## Stack
- NestJS
- React
- lightweightcharts
- Docker

## How to start

## Using Docker
``` bash
$ docker-compose build
$ docker-compose up
```

## Without docker
- In /fats-down-server
``` bash
npm run start
```
- In /fats-down-client
``` bash
npm run start
```
